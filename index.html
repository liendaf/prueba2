HTML5

Es la última versión del lenguaje de marcado. Usa etiquetas como <header>, <main>, <footer> para dar estructura semántica.

CSS3

Permite definir estilos visuales como colores, tamaños y distribución de los elementos en pantalla.

Curiosidades

HTML significa "Lenguaje de Marcado de Hipertexto".

CSS permite separar el contenido del diseño.

La etiqueta <aside> se usa para contenido relacionado, como notas o enlaces.
